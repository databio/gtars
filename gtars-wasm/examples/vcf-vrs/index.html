VCF → VRS allele identifiers (browser harness)

In-browser VCF→VRS tool with zero server compute. The reference genome is downloaded once and cached in your browser; every variant is converted to a ga4gh:VA.<digest> identifier entirely on your machine. Nothing is sent to a server.

1. Reference genome

The URL of a hosted reference genome store. Verifying only reads a small index; each chromosome is downloaded the first time a dropped VCF needs it, then kept in your browser's local storage so later runs are instant.
