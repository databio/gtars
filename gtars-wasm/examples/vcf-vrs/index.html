HGVS / VCF → VRS allele identifiers

In-browser tool to convert variants — a whole VCF file or a list of HGVS expressions — into GA4GH ga4gh:VA.<digest> identifiers, with zero server compute. The reference genome is downloaded once and cached in your browser; everything runs on your machine.

1. Reference genome

The URL of a hosted reference genome store. Verifying only reads a small index; each chromosome is downloaded the first time a dropped VCF needs it, then kept in your browser's local storage so later runs are instant.
